# Лямбда-выражения
---
### Что такое лямбда-выражение? Чем его можно заменить?      
Лямбда-выражение - упрощённая запись анонимного класса, реализующего функциональный интерфейс

### Что такое ссылка на метод?
Ссылка на статический метод - ContainingClass::staticMethodName  
   
Ссылка на нестатический метод конкретного объекта - containingObject::instanceMethodName  
   
Ссылка на конструктор - ClassName::new  
  
**Ссылка на метод** - это сокращенный синтаксис выражения лямбда, который выполняет только один метод. Это позволяет нам ссылаться на конструкторы или методы, не выполняя их.

### Может ли лямбда-выражение быть в несколько строк?
Да (тогда нужно заключить тело лямбды в фигурные скобки использовать точку с запятой между выражениями и return)

### Любой анонимный класс можно заменить на лямбду?
Нет, только тот который содержит один метод  
Только анонимные классы, которые являются реализациями интерфейса SAM (например, Runnable, ActionListener, Comparator, Predicate), могут быть заменены выражением лямбда.

### К каким переменным и как можно обращаться в теле лямбда-выражений?
* Переменные внутри метода (эффективно-финальные)
* static переменные класса
* переменные интерфейса (с которым лямбда работает)
